# Consumo de API Pública - PokéDex

Este es un proyecto académico que consiste en una página web que consume datos de una API pública de manera dinámica empleando buenas prácticas de JavaScript moderno.

## Características de la Aplicación
* **Estructura Semántica**: Construida bajo estándares de HTML5.
* **Diseño Responsivo**: Grid Layout y Flexbox adaptables a dispositivos móviles.
* **Consumo de API**: Implementación de llamadas asíncronas con `async/await` y `fetch`.
* **Manipulación de Arreglos**: Empleo riguroso de `.map()` y `.forEach()`.
* **Inyección Dinámica**: Tarjetas generadas en su totalidad desde el árbol DOM con JavaScript.

## Información de la API Utilizada
* **Nombre de la API**: PokéAPI
* **URL de la API**: https://pokeapi.co/
